Vessel Trader Game

Summary

  • 3d first person airship piloting game set in a civilization-less world with just trade hubs and few POI to populate it - low poly art style
  • ship module upgrade system when inside a hangar
  • mobile first, pc port second
  • systems
    • world navigation
      • 2d map
      • weather events like storms, thunder strikes, acid rain, etc can spawn on world map and move in a certain direction, when pin inside event area environment changes drastically
    • airship piloting
      • locally loaded 3d scenes
    • event managers
      • poi generation
      • enemy airship generation
    • weapons
    • hull segments/durability??????
    • crew ordering with commander menu and task manager?
    • inventory system
    • economy system?
  • main gameplay loop: trade wares at outposts, equip vessel with upgrades, chart out next route, top up vessel with ammo, set off to the next trade hub
  • journeys consist of a series of pins on a world map. reaching a pin launches an event chosen randomly from a pool of events for the region the pin is placed in
    event types:
    • foreign ships can appear
      • enemies like pirates will attack
      • merchants may want to trade
      • abandoned dead ships can be looted
    • points of interest with loot

Controls

  • Everything gets done from within the cockpit, no movement inside airship - maybe locked camera as well or restricted look-around
  • Helm for controlling airship rotation and pitch with lever for speed, both have deadzones in center
    • Mobile: tilt controls or dragging helm joystick with finger (needs setting for selecting control, also add virtual helm joystick in corner); drag speed lever with finger
    • PC: dragging helm joystick with mouse, dragging speed lever with LMB or scrolling during hover
  • Communicator for sending out crew orders or initiating trades
    • Mobile: tap opens symbol options, taps open submenus until definitive command has been selected, clicking communicator again hides menu
    • PC: same as mobile but with LMB instead
  • Weapon Wheel for selecting one weapon to shoot with
    • Mobile: swipe left or right to select previous or next installed weapon
    • PC: scrolling while hovering over or clicking half visible area on adjacent wheel segment
  • Shoot selected weapon
    • Mobile: button tap/hold w cooldown
    • PC: RMB
  • See important information on status monitor
    • Mobile: tap on monitor to open, has buttons for switching screens
    • PC: LMB on monitor

Weapons

  • weapons are organized into categories and quality tiers

    • auto cannon
      • fires bullets at a constant speed, rate and damage
    • rail gun
      • charging up a shot for longer increases damage
    • rocket launcher
      • launches homing missiles that fly to the heat signature closest to their looking direction in their view cone (flares also count), gives explosion and high damage, rather long cooldown
    • mine dispenser
      • drops mines that float back up which explode when near any object or when taking damage
    • gatlin
      • starts slow, gets faster
      • stream of bullets, less heat per bullet than autocannon but more heat per second
    • flak
      • single explosive rounds, straight line, faster rate than rockets
    • shotgun
      • burst of projectiles
    • harpoon
      • shoot at object to create tether that can be released, pulling the object closer
      • least heat generation
    • flare
      • irritates homing missiles
    • burst turbine
      • releases a wind burst that pushes objects away

  • each weapon has random stats of its category whose ranges are defined by the quality tier

Challenges

Special challenge quests can be taken that can include racing to a distant place within a time limit, taking on some enemies, or trying to obtain rare items.

Auto Pilot mode

  • direct vessel upgrade (early-mid game)
  • capable of piloting the vessel across the map to next pin without spawning an event
  • usecase
    • have active pins on map
    • activate the autopilot
    • wait for the ship to move from pin to pin directly

System Requirements

1. Map Navigator

  • allows adding up to a predefined number of pins onto the map within a certain distance of another. Pin reach can be modified during the game. Pin placement order needs to be saved.
  • if at least one pin is active, character can move towards the next placed pin when receiving command to do so, emitting a signal when reaching that pin
    Extra control: navigate
  • launches command to world navigator to navigate to next pin
    • lock ship controls
    • fog up windows
    • preload next pin event
    • upon reaching reaching next pin: if autopilot disabled, launch preloaded event, else go directly to next pin

2. Communicator

Clickable object that carries a hierarchical tree of commands and categories. when clicking a command it activates the command and hides the full tree. Clicking a category will hide the current tree layer and show its own children instead.

3. Ship maneuvering

  • three controls:
    • control joystick for ship tilt, stays in position outside of deadzone
    • thrust lever for engine power, forward as well as backward
    • engine power button, hold to start engine tap to kill engine

4. Segmented hull integrity

5. Weapon firing

6. Environment manager

  • split into terrain and enemy manager
    Terrain manager
  • spawns in points of interest that carry loot and do nothing else
  • player needs to approach poi carefully to not take collision damage
  • when in reach, can order crew to loot poi - speed depends on pois size and number of crew members assigned
    Enemy manager
  • spawns in enemy ships that can attack the player using their mounted weapons and if killed can be looted for goods
  • enemy ships are prebuilt at first, later generated randomly with ship generator
    Ship generator:
  • starts with gondola
  • adds cockpit and gasbag
  • adds engines

7. Trade Hubs

8. Crew task manager

  • list of crew members that each have a list of different abilities as well as an effectiveness multiplier per ability
  • every crew member can perform one task at a time, starting immediately after being commanded to
  • crew members can perform different tasks that each have have name, icon, duration and a completion signal that goes off after task has been completed
  • task duration is not counted in seconds but in abstract number, each delta frame 1 crew member subtracts {efficiency} from duration number, when hits 0 task is completed
  • extra page on status monitor that shows current list of tasks and how far they are completed

Route Pinning:

The map allows players to set pins anywhere within a radius around the players current position, distance depends the pins upgrade level (upgraded by acculating higher ranking achievements). When a pin is set, the next pin can be attached within a radius around the last pins position. Next pin is the one with the largest radius available. Pins can be removed by right clicking.
Outposts can be pinned to approach them for trading, upgrading and other harbour based tasks.
